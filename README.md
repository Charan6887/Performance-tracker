# Performance Tracking Website

## Description
- This is a web application to track and visualize performance of our employees. It allows adding, updating, and monitoring performance data over time.
- HR Uses performance scores for decisions (salary, promotion, support)
- Manager Assigns the task to employee.
- Gives rating and feedback on the employee performance towards the task/Target an individual completed.
- That feedback and rating is used to calculate the score.
- Based on the score HR makes the decision regarding the employee

## Note: Only HR HAS ACCESS TO CREATE THE EMPLOYEES PROFILE


## Features
- Add and edit Tasks assigned by manager
- Employees can view and update the progress of the task assigned to them individually
- User-friendly interface

## Panels
- Manager Login panel
- Employee Login Panel
- HR Login Panel 

## Technologies Used
- HTML, CSS, JavaScript
- Php
- SQL(DATABASE)
